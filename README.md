# 🧮 BMI Calculator (Casual Python Project)

Hey there! 👋  
This is a simple **Body Mass Index (BMI) Calculator** made with Python.  
It helps you find out whether you're underweight, normal, overweight, or obese — all in a fun and casual way! 😄

---

## 🚀 Features
- Takes user input for **weight (kg)** and **height (meters)**
- Calculates BMI using the standard formula  
  \[
  BMI = \frac{weight}{height^2}
  \]
- Gives an easy-to-understand health category message  
- Handles invalid input gracefully with an error message

---

## 🧰 Requirements
You just need **Python 3** installed on your computer.  
No external libraries are required.
